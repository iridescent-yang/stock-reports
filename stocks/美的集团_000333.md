## 二、🌳 产业认知地图

### 它到底是干嘛的？

> 从你家客厅的空调，到特斯拉工厂里的库卡机械臂，到数据中心的热管理系统——美的已经不是一个"卖家电的"。它更像一个**以家电为现金奶牛、不断孵化新业务的科技控股平台**。

### 三层业务结构

```
第一层：Cash Cow（现金奶牛，66%）
  空调（市占率37.7%第一）+ 冰箱/洗衣机/小家电
  9个品类全国市占率第一
  → 每年贡献3000亿+收入，利润率稳定

第二层：Growth Engine（增长引擎，～20%）
  新能源车热管理 + 储能 + 工业电机
  → 跟着新能源车增长，年增速15-20%

第三层：Future Option（未来期权，～14%）
  库卡机器人 + 楼宇科技 + 智慧医疗
  → 工业4.0/AI机器人，成功率不确定但想象空间大
```

### 关键产业变动

**以前：** 美的 = 空调 + 冰箱 = 跟着地产周期走，PE永远10倍出头。

**现在正在改变的：**
1. **全球化**：海外收入从28%→43%，东南亚/中东/拉美建厂，不再只卖给中国人。
2. **ToB脱胎换骨**：库卡扭亏为盈，新能源热管理年增速20%+。家电外衣下藏着工业科技公司。
3. **高股息回馈**：分红+回购合计约350亿，股息率5.3%。千亿市值里排前五。
