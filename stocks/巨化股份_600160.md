## 二、🌳 产业认知地图

### 它到底是干嘛的？

> 你家空调夏天吹冷风、冰箱冻冰块——背后是**制冷剂**在干活。它是一种液体→气体反复循环的"热能搬运工"。巨化股份不仅做制冷剂，还从最上游的萤石矿挖起，一路做到半导体工厂用的超高纯含氟材料。

### 应用场景演进

| 时代 | 谁在用 | 需求特征 |
|------|--------|----------|
| 过去 | 家用空调、冰箱 | 量大但增长见顶，竞争激烈 |
| 现在 | 汽车空调+工业制冷 | 新能源车每台热泵用制冷剂量是传统车2-3倍 |
| 未来 | AI数据中心液冷+半导体蚀刻 | 液冷需要特种制冷剂；子公司中巨芯供G5级电子氢氟酸给晶圆厂 |

### 关键产业变动

**配额制锁死供给——这行业已经不是谁都能进的。**

- 2024年起，三代制冷剂（HFCs）生产配额全球冻结
- 中国按企业历史产量分配配额，**以后只减不增**
- 只有拿到配额的企业能生产，新进入者永久被挡在门外
- 巨化拿到39.30%，一骑绝尘

这就相当于：政府给制冷剂行业发了"特许经营牌照"，而且只发给少数几家。**需求在涨（空调换新+新能源车+AI液冷），供给被锁死了——价格只会涨。**

### 产业生命周期

```
📈 当前位置：
萌芽 ──→ 成长 ──→【繁荣早期】──→ 成熟 ──→ 衰退
                  ↑
              我们现在在这
              
驱动力（确定性极强）：
  1. 配额锁死（2024年冻结，之后只减不增）
  2. 空调换新+新能源车热泵（需求年增5-8%）
  3. AI液冷（新增量，拉动高端制冷剂）
  4. 半导体材料（PFA/FFKM国产替代，打开第二增长曲线）
  
风险点：
  - 第四代制冷剂（HFOs）长期可能替代三代，但3-5年内成本高、产能极小
  - 空调销售受天气和地产周期影响
```

### 直觉验证

洋哥哥，你现在感受到的这些趋势都在印证制冷剂行业：
- 夏天越来越热 → 空调卖得更多 → 制冷剂需求更大 ✅
- 路上新能源车越来越多 → 每台都要热泵 → 2-3倍制冷剂用量 ✅
- AI数据中心越建越多 → 液冷需要特种制冷剂 → 新需求 ✅

**打开手机天气预报，"今年高温天越来越多"就是对这个行业最直接的验证。**
